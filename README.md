# SageCoach – Virtual Fitness & Physical Exam Training Platform

## Overview

SageCoach is a virtual coaching platform designed to help individuals prepare for government physical fitness examinations and improve their overall physical health.

The platform provides structured training guidance for government exam aspirants who need to pass physical fitness tests, while also supporting general fitness users such as gym enthusiasts and yoga practitioners.

SageCoach acts as a digital coach that helps users stay disciplined, track progress, and follow proper training routines.

---

## Problem Statement

Many government job aspirants struggle to prepare for the physical fitness tests required in recruitment exams. Most candidates lack proper guidance, structured workout plans, and training discipline.

At the same time, beginners in fitness and yoga often do not know how to start training effectively.

SageCoach aims to solve this problem by providing a simple digital platform that guides users through structured physical training.

## Installation

### Clone Repository

```
git clone https://github.com/priyanshkamlesh/sageCoach.git
```

### Install Backend Dependencies

```
cd backend
pip install -r requirements.txt
```

### Run Backend

```
python apiserver.py
```

### Run Frontend

```
cd frontend
npm install
npm run dev
```
